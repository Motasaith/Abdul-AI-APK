## üõ†Ô∏è Development Setup

If you're a developer and want to build or contribute to Abdul AI, follow these steps to set up the project locally.

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [Expo Go](https://expo.dev/client) app on your mobile device (or an Android/iOS emulator)

### Getting Started

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Motasaith/Abdul-AI-APK.git
    cd Abdul-AI-APK
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npx expo start
    ```

4.  **Run on your device**
    - Scan the QR code with the **Expo Go** app (Android) or Camera app (iOS).
    - Press `a` to run on Android Emulator or `i` to run on iOS Simulator.
